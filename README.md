# Vibe catcher


Веб-приложение на базе искусственного интеллекта и современных методов обработки естественного языка (NLP) для автоматического сбора, анализа и визуализации отзывов о брендах. Проект позволяет бизнесу мгновенно взглянуть на свой продукт «глазами потребителя», выявить сильные и слабые стороны, а также автоматизировать рутинную работу маркетологов и аналитиков.


## Области применения


* Управление репутацией бренда и PR: позволяет PR-отделам вовремя замечать всплески негатива, изолировать репутационные кризисы на ранней стадии и точечно отрабатывать жалобы клиентов.
* Продуктовая аналитика и QA: аналитики могут по ключевым словам из негативных отзывов понять,что чаще всего ломается или не нравится пользователям.
* Competitive Intelligence: анализ сильных и слабых сторон прямых конкурентов. Введя в систему бренд соперника, компания может найти их уязвимые места (например, плохой сервис или долгая доставка) и использовать это в своей маркетинговой стратегии.
* Customer Success и поддержка: помогает выгружать отсортированные списки критических отзывов для ручной обработки отделом заботы о клиентах, что поможет превратить недовольных пользователей в лояльных.


## Ключевые возможности

* Автоматический парсинг данных: динамический сбор актуальных отзывов по названию бренда с целевой платформы.
* Sentiment Analysis: глубокая оценка контекста отзывов и их классификация на позитивные, негативные и нейтральные с помощью нейросети RuBERT.
* Интерактивный дашборд: наглядная визуализация общей аналитики бизнеса (общее количество отзывов, процентное соотношение тональностей на круговой диаграмме).
* Качественный анализ текста: выделение сильных и слабых сторон бренда на основе лемматизации и фильтрации текста.
* Экспорт данных: возможность скачать структурированную таблицу со всеми отсортированными отзывами для дальнейшего ручного анализа.

## Стек технологий

* Интерфейс: Streamlit
* Парсинг данных: Playwright (для динамического контента), BeautifulSoup4
* Обработка языка:модель RuBERT для классификации тональности, PyMorphy3 (для морфологического анализа и лемматизации текста)
* Анализ данных: Pandas
  

## Установка

1. Клонируйте репозиторий:


git clone https://github.com/The-Anna-BAE/vibe-catcher.git
cd vibe-catcher



2. Создайте и активируйте виртуальное окружение:


python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate



3. Установите необходимые зависимости:


pip install -r requirements.txt



4. Установите системные браузеры для корректной работы библиотеки Playwright:


playwright install


## Использование

Проект разделен на два основных модуля: скрипт для парсинга данных и интерактивный веб-интерфейс. Вся система запускается через консоль одной командой.

Для запуска веб-приложения выполните в терминале команду streamlit run, указав путь к основному файлу интерфейса:


streamlit run website_analyse.py


### Как работать с интерфейсом:

1. После запуска в браузере откроется вкладка с приложением.
2. Введите название интересующего вас бренда в поисковую строку.
3. Система автоматически запустит фоновый парсер, соберет отзывы, проведет их очистку, лемматизацию и отправит в нейросеть для анализа тональности.
4. На экране отобразится круговая диаграмма (Pie Chart) с соотношением позитива/негатива, а также словари ключевых сильных и слабых сторон.
5. При необходимости воспользуйтесь кнопкой «Скачать таблицу», чтобы сохранить результаты в формате CSV для ручной проверки.

## Структура проекта

* website_analyse.py - основной файл веб-интерфейса Streamlit, NLP-инференс и отрисовку графиков.
* analysis.py - файл с кодом, отвечающий за логику приложения. 
* another_scrapper.py - скрипт парсера, использующий Playwright и BeautifulSoup для извлечения данных с сайтов.
* requirements.txt - файл со списком всех внешних библиотек и зависимостей проекта.

## Модель

Для анализа русскоязычного текста используется предобученная языковая модель RuBERT. Распознает сарказм, скрытый негатив и контекстные особенности отзывов реальных клиентов, обеспечивая высокую точность бизнес-метрики.
